# Phase 08 — 업로드 + 미리보기(서버 액션)

**목표:** 엑셀 업로드 → 미리보기(신규/갱신/스킵/경고), 확정 전 단계.
**규약:** `CLAUDE.md` 6번, 7번(서버 전용 키).

## 명령
```
admin/subscriptions에 '엑셀 업로드' 영역을 추가해줘. 파일 선택 → 서버 액션에서 parseRoster 실행
→ 신규/갱신/스킵 건수와 경고를 표로 미리보기. '확정'을 눌러야 DB 반영.
연도는 사용자가 선택(예: 2025). service_role 클라이언트는 서버 액션 안에서만 사용.
```
